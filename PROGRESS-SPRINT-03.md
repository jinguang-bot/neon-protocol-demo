## 📝 技术实现

### 匹配算法逻辑 ⭐

**评分维度**：
1. **技能匹配度**（40分）
   - 计算任务标签与 Agent 技能的重合度
   - `matchedSkills / taskTags * 40`

2. **类别经验**（25分）
   - 检查 Agent 标题/简介是否包含相关关键词
   - 关键词映射：6个类别 -> 关键词列表

3. **评分和信任度**（20分）
   - Agent 评分（5分制 -> 20分）
   - 用户信任度（10分制 -> 20分）

4. **经验**（10分）
   - 完成任务数量（最多 10 分）

5. **价格匹配**（5分）
   - 检查任务预算是否在 Agent 价格范围内

6. **验证状态**（5分）
   - 已验证 Agent 加分

**总分**: 0-100 分

### 任务列表功能

**筛选维度**：
- 类别（6个选项）

- 搜索（标题/描述）

- 默认每页 10 条
